# 알고리즘반 6주차 일지
### 개요
- 일자: 2023년 8월 16일 19시
- 방식 : 비대면 진행(ZOOM)
- 주제: 최단경로(다익스트라/플로이드-와샬)
- 수업 내용 :
- 트리/신장트리/최소신장트리 개념 설명
- MST를 구하는 두 알고리즘(프림, 크루스칼) 소개
- 크루스칼 동작 원리 설명 및 직접 구현
- DAG 개념 설명
- 위상정렬을 구현하는 두 가지 방법 소개(진입차수, 진출차수)
- 진입차수와 큐를 이용하는 구현 방법 설명 및 직접 구현
